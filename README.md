# react-native-story-view
-------------
#### Steps to Run & Build:

- Install dependencies ```yarn```
- Build ```yarn build```
- Install dependencies in example app ```cd example && yarn && cd ios/ && pod install && cd ..```
- Run example app ```yarn ios```
-----
- Run ```yarn build``` to sync package changes / updates
